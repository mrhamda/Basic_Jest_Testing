# üïã Next.js Prayer Times (Salah) Tracker

A modern, responsive web application that fetches real-time prayer timings using the My-Masjid API. Built with **Next.js 15**, **GSAP** for smooth animations, and **Tailwind CSS**.

## ‚ú® Features

* **Real-time Fetching:** Server-side data fetching with revalidation every hour.
* **Automatic Highlights:** Highlighting the current/next prayer time based on the user's local clock.
* **Countdown Timer:** Precise "time remaining" calculation (rounded up for accuracy).
* **Animated UI:** Smooth entry animations using GSAP and ScrollTrigger.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Friday Logic:** Automatically switches from "Dhuhr" to "Jummah" on Fridays.

* **API:** My-Masjid Public API
